# megaptera-front

- 교육 기간: `2023.1.30` ~ `2023.04.23`
- 메가테라 프론트엔드 생존코스의 내용을 기록합니다.

## 구현 기능

- 서비스

  - 상품 보기
  - 상점 보기
  - 상품 옵션
  - 장바구니
  - 주문
  - 회원가입, 로그인

- 어드민
  - 상품 관리
  - 상점 관리
  - 상품 옵션 관리
  - 주문 관리
  - 회원 관리

## 다루는 기술

- TypeScript
- JSX
- state-view with React
- Component
- Flux Architecture
- separation of concerns
- usestore-ts
- Fetch API, CORS
- Express
- Jest, MSW
- CSS-in-JS
- Design System
